# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Empezar a trabajar con el Portal de confianza del servicio de Microsoft

El portal del servicio de confianza a Microsoft proporciona una gran variedad de contenido, herramientas y otros recursos sobre prácticas de seguridad, privacidad y cumplimiento de Microsoft.
  
## <a name="accessing-the-service-trust-portal"></a>Acceso al Portal de confianza del servicio

El Portal de servicio de confianza contiene detalles sobre la implementación de Microsoft de controles y procesos que protejan los datos del cliente y nuestros servicios de nube en él. Para obtener acceso a algunos de los recursos en el Portal de servicio de confianza, debe iniciar sesión como un usuario autenticado con su cuenta de servicios de nube de Microsoft (una cuenta de organización de Azure Active Directory o una Account de Microsoft) y revise y acepte el Microsoft Acuerdo de confidencialidad para materiales de cumplimiento.
  
**Clientes existentes pueden obtener acceso al Portal de confianza de servicio** en [https://aka.ms/STP](https://aka.ms/STP) con una de las siguientes suscripciones en línea (versión de prueba o de pagadas): 
  
- Office 365

- Dynamics 365

- Azure
    
 > [!NOTE]
 > Azure cuentas de Active Directory asociadas con las organizaciones tienen acceso a toda la gama de características como el Administrador de cumplimiento de normas y documentos. Cuentas de Microsoft creadas para su uso personal tienen acceso limitado a contenido de Portal de servicio de confianza. 
  
**Nuevos clientes y los clientes de evaluación de Microsoft online services**
  
Para crear una nueva cuenta o para crear una cuenta de prueba, use uno de los siguientes formularios de inscripción (también se utiliza para las cuentas de prueba) para obtener acceso a la STP.
  
- Registrarse y obtener una nueva [cuenta de prueba de Office 365 empresarial](https://go.microsoft.com/fwlink/p/?LinkID=507653) o una nueva [cuenta de prueba de Office 365 Enterprise](https://go.microsoft.com/fwlink/p/?LinkID=698279)

- Registrarse y obtener una nueva [cuenta de prueba de Dynamics 365](https://go.microsoft.com/fwlink/?LinkId=252780)

- Regístrese para obtener una nueva [cuenta de prueba de Azure](https://go.microsoft.com/fwlink/?LinkId=722737). 
    
Al registrarse para una versión de prueba gratuita o una suscripción, debe habilitar Azure Active Directory admitir el acceso a la STP.
  
## <a name="using-the-service-trust-portal"></a>Uso del Portal de confianza del servicio

Las características de Portal de servicio de confianza y contenido son accesibles desde el menú principal.
  
![Menú de Portal de servicio de confianza](media/STPMenus1.png)

Las secciones siguientes describen cada elemento en el menú principal.
  
### <a name="service-trust-portal"></a>Portal de confianza del servicio

El vínculo de **Portal de servicio de confianza** , muestra la página principal. Proporciona una manera rápida de volver a la página principal.

### <a name="compliance"></a>Cumplimiento

Proporciona las siguientes herramientas relacionadas con el cumplimiento de normas y recursos:

- Se muestra de **informes de auditoría** - una lista de informes de auditoría y evaluación de independientes en los servicios de nube de Microsoft. Estos informes proporcionan información acerca de Microsoft Cloud services cumplimiento de normas de protección de datos y los requisitos legales, tales como:
  
    - Organización internacional de normalización (ISO)
    - Controles de la organización de servicio (seguridad social)
    - National Institute of Standards and Technology (NIST)
    - Riesgo federal y el programa de administración de la autorización (FedRAMP)
    - Reglamento de protección de datos general (GDPR)

  También se muestra información sobre los informes de auditoría y un vínculo a la página **informes de auditoría** en la página principal de Portal de servicio de confianza. 
  
- **Administrador de cumplimiento de** -Compliance Manager es una herramienta de evaluación de riesgos basada en flujo de trabajo que le ayuda a realizar un seguimiento, asignar y comprobar las actividades de cumplimiento de las normativas de la organización relacionadas con los servicios de Microsoft Cloud, como Office 365, Dynamics 365, y Azure. Use el Administrador de cumplimiento para administrar el cumplimiento de normativas dentro del modelo de responsabilidad compartida de la nube. Para obtener más información, consulte [Use el Administrador de cumplimiento de normas para ayudar a satisfacer requisitos legales cuando se usa servicios de nube de Microsoft y protección de datos](meet-data-protection-and-regulatory-reqs-using-microsoft-cloud.md).

### <a name="industries--regions"></a>Sectores & regiones

Proporciona información de cumplimiento de normas específicas de la industria y el área acerca de los servicios de Microsoft Cloud.

- **Sectores** - proporciona páginas de aterrizaje específicas de la industria para obtener información y sobre cómo Microsoft Cloud services ayudar a la organización que sea compatible con los estándares y regulaciones de estas industrias:

    - Education
    - Servicios financieros
    - Gobierno
    - Mantenimiento
    - Industria
    - Comercial

- **Regiones** - proporciona opiniones legales en cumplimiento de servicios de Microsoft Cloud con diversos las leyes de distintos países. Ciertos países incluyen Australia, República Checa, Alemania, Polonia, Rumania, España y el Reino Unido.
  
### <a name="documents--resources"></a>Recursos de & de documentos

Proporciona una gran cantidad de información sobre el diseño e implementación de seguridad con el objetivo de facilitar cumplir los objetivos de cumplimiento de normas mediante la descripción de cómo Microsoft Cloud services mantienen sus datos seguros. Para revisar el contenido, seleccione una de las opciones en la página **documentos & recursos** .

- Notas del producto, las preguntas más frecuentes y artículos-sobre
- Guías de cumplimiento
- Pruebas de lápiz y las evaluaciones de seguridad
- Puntuación de seguro
- Seguridad de Azure y planos técnicos de cumplimiento
- Controles auditados
  
### <a name="trust-center"></a>Centro de confianza

Vínculos al [Centro de confianza de Microsoft](https://www.microsoft.com/trustcenter), que proporciona más información acerca de la seguridad, el cumplimiento y la privacidad en el Microsoft Cloud. Esto incluye información acerca de las capacidades en servicios de Microsoft Cloud que puede usar para satisfacer los requisitos específicos de la GDPR, documentación útil a su responsabilidad GDPR y la descripción de las medidas técnicas y de organización Microsoft ha adoptado para admitir la GDPR.
  
### <a name="my-library"></a>Mi biblioteca

Esta nueva función permite que guardar (o *pin*) de documentos para que rápidamente se pueden tener acceso a ellas en la página Mi biblioteca. También puede configurar las notificaciones para que Microsoft envía un mensaje de correo electrónico cuando se actualizan los documentos en la biblioteca de mi. Para obtener más información, vea la sección [Mi biblioteca](#my-library-1) en este artículo.

### <a name="admin"></a>Administrador

Funciones administrativas que sólo están disponibles para la cuenta de administrador global. Esta opción sólo es visible cuando haya iniciado sesión como administrador global.
  
 La página de **configuración** permite para asignar al administrador de cumplimiento de acceso basado en roles. Para obtener más información, vea la sección "Permisos y control de acceso basado en roles" en el [Administrador de cumplimiento de normas de uso para ayudar a satisfacer requisitos legales cuando se usa servicios de nube de Microsoft y protección de datos](meet-data-protection-and-regulatory-reqs-using-microsoft-cloud.md#permissions-and-role-based-access-control).
  
### <a name="search"></a>Búsqueda

Haga clic en la lupa en la esquina superior derecha de la página de Portal de servicio de confianza para expandir el cuadro, escriba los términos de búsqueda y presione **ENTRAR**.

![Portal de confianza de servicios: Campo de búsqueda](media/7c5cd817-3d62-420b-adb4-76e33fef941f.png)
  
 Se muestra la página de **búsqueda** , con el término de búsqueda que se muestra en el cuadro de búsqueda y los resultados de búsqueda que se enumeran a continuación.
  
![Portal de confianza de servicios: Buscar en documentos con filtro aplicado](media/86b754e1-c63c-4514-89ac-d014bf334140.png)

De forma predeterminada, la búsqueda devuelve los resultados de documento. Los resultados se pueden mediante el uso de listas de la lista desplegable para perfeccionar la lista de documentos que se muestran. Puede usar varios filtros para reducir la lista de documentos. Filtros incluyen los servicios de nube específicos, categorías de cumplimiento de normas o prácticas de seguridad, regiones y sectores. Haga clic en el vínculo del nombre de documento para descargar el documento.
  
En la lista de controles desde evaluaciones en el Administrador de cumplimiento de normas relacionadas con los términos de búsqueda, haga clic en **Administrador de cumplimiento**. Los resultados de búsqueda muestran la fecha de la evaluación se ha creado, el nombre de la evaluación de agrupación, el servicio de Microsoft Cloud aplicable, y si el control es de Microsoft o administradas por el cliente. Haga clic en el nombre de un control para ver el control en la evaluación en el Administrador de cumplimiento de normas.
  
![Portal de confianza de servicios: Búsqueda en controles del Administrador de cumplimiento](media/bafb811a-68ce-40b5-ad16-058498fe5439.png)
  
> [!NOTE]
> Informes de Portal de servicio de confianza y documentos están disponibles para descargar para al menos 12 meses después de la publicación o hasta que esté disponible una nueva versión del documento.
  
## <a name="my-library"></a>Mi biblioteca

Usar la característica de mi biblioteca para agregar recursos y documentos en el Portal de servicio de confianza a la página Mi biblioteca. Esto le permite tener acceso a documentos que son relevantes para usted en un solo lugar.  Para agregar un documento a la biblioteca de mi, haga clic en el menú **...** a la derecha de un documento y, a continuación, seleccione **Agregar a la biblioteca**. Puede agregar varios documentos a la biblioteca de mi haciendo clic en la casilla de verificación situada junto a uno o más documentos y, a continuación, haga clic en **Guardar en la biblioteca** en la parte superior de la página.

Además, la característica de notificaciones permite configurar la biblioteca de mi de modo que se envía un mensaje de correo electrónico para cuando Microsoft actualiza un documento que ha agregado a la biblioteca de mi. Para configurar las notificaciones, vaya a la biblioteca de mi y haga clic en **Configuración de la notificación**. Puede elegir la frecuencia de las notificaciones y especificar una dirección de correo electrónico de la organización para enviar las notificaciones. Notificaciones de correo electrónico incluyen vínculos a los documentos que se han actualizado y una breve descripción de la actualización.

Tenga en cuenta también que se identifiquen todos los documentos en la biblioteca de mi que se han actualizado en los últimos 30 días, independientemente de si o no activar las notificaciones. Una breve descripción de la actualización también se muestra en una información sobre herramientas. 

## <a name="starter-packs"></a>Paquetes para principiantes

Los paquetes de inicio son un conjunto de curated Microsoft de documentación acerca de los servicios de Microsoft Cloud para sectores específicos. Actualmente, el Portal de servicio de confianza ofrece los siguientes paquetes de tres starter para las organizaciones de servicios financieros. Estos paquetes de inicio ayudan a las organizaciones a evaluar y evaluar la seguridad, el cumplimiento y la privacidad en el Microsoft Cloud y se proporcionan instrucciones para ayudar a implementar Microsoft Cloud services en la industria de servicios financieros altamente regulado.

- **Paquete de inicio de evaluación** - uso para la evaluación temprana de la nube de Microsoft para las organizaciones de servicios financieros.

- **Evaluación Starter Pack** - después de la evaluación, use las listas de comprobación y otras instrucciones en este módulo de inicio para ayudar a su organización a evaluar los riesgos relacionados con la seguridad, el cumplimiento y la privacidad.

- **Auditoría Starter Pack** - usuario este pack de inicio para obtener instrucciones sobre el uso de controles de auditoría y otras herramientas para ayudarle a su implementación de servicios de Microsoft Cloud de manera que ayuda a reducir la exposición de la organización a los riesgos.

Para obtener acceso a estos paquetes para principiantes, vaya al **Portal de servicio de confianza > sectores & regiones > soluciones de la industria > servicios financieros**. Puede abrir o un documentos descarga desde un starter pack o guardarlos en la biblioteca de mi.
