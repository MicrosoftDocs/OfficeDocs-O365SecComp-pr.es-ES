<span data-ttu-id="89a7e-p102">Azure ofrece varios mecanismos para proteger los datos a medida que se mueven de una ubicación a otra. Perfect Forward Secrecy (PFS) protects connections between customers' client systems and Microsoft's cloud services by unique keys. Connections also use RSA-based 2,048-bit encryption key lengths. This combination makes it difficult for someone to intercept and access data that is in-transit.</span></span>

<span data-ttu-id="89a7e-p103">Los datos se pueden proteger en el tránsito entre una aplicación y Azure mediante [cifrado del lado cliente](https://docs.microsoft.com/azure/storage/storage-client-side-encryption), https o SMB 3,0.
