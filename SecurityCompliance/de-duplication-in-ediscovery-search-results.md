## <a name="how-duplicate-messages-are-identified"></a><span data-ttu-id="60897-110">Identificación de los mensajes duplicados</span><span class="sxs-lookup"><span data-stu-id="60897-110">How duplicate messages are identified</span></span>

<span data-ttu-id="60897-111">Las herramientas de eDiscovery de Office 365 usan una combinación de las siguientes propiedades de correo electrónico para determinar si un mensaje es un duplicado:</span><span class="sxs-lookup"><span data-stu-id="60897-111">Office 365 eDiscovery tools use a combination of the following email properties to determine whether a message is a duplicate:</span></span>
  
- <span data-ttu-id="60897-112">**InternetMessageId** : esta propiedad especifica el identificador de mensaje de Internet de un mensaje de correo electrónico, que es un identificador único global que hace referencia a una versión específica de un mensaje específico.</span><span class="sxs-lookup"><span data-stu-id="60897-112">**InternetMessageId** - This property specifies the Internet message identifier of an email message, which is a globally unique identifier that refers to a specific version of a specific message.</span></span> <span data-ttu-id="60897-113">Este identificador lo genera el programa cliente de correo electrónico del remitente o el sistema de correo electrónico de host que envía el mensaje.</span><span class="sxs-lookup"><span data-stu-id="60897-113">This ID is generated by the sender's email client program or host email system that sends the message.</span></span> <span data-ttu-id="60897-114">Si una persona envía un mensaje a más de un destinatario, el identificador del mensaje de Internet será el mismo para cada instancia del mensaje.</span><span class="sxs-lookup"><span data-stu-id="60897-114">If a person sends a message to more than one recipient, the Internet message ID will be the same for each instance of the message.</span></span> <span data-ttu-id="60897-115">Las revisiones posteriores del mensaje original recibirán un identificador de mensaje diferente.</span><span class="sxs-lookup"><span data-stu-id="60897-115">Subsequent revisions to the original message will receive a different message identifier.</span></span> 
    
- <span data-ttu-id="60897-116">**ConversationTopic** : esta propiedad especifica el asunto de la secuencia de conversación de un mensaje.</span><span class="sxs-lookup"><span data-stu-id="60897-116">**ConversationTopic** - his property specifies the subject of the conversation thread of a message.</span></span> <span data-ttu-id="60897-117">El valor de la propiedad **ConversationTopic** es la cadena que describe el tema general de la conversación.</span><span class="sxs-lookup"><span data-stu-id="60897-117">The value of the **ConversationTopic** property is the string that describes the overall topic of the conversation.</span></span> <span data-ttu-id="60897-118">Una conservación consta de un mensaje inicial y de todos los mensajes que se envían como respuesta al mensaje inicial.</span><span class="sxs-lookup"><span data-stu-id="60897-118">A conservation consists of an initial message and all messages sent in reply to the initial message.</span></span> <span data-ttu-id="60897-119">Los mensajes dentro de la misma conversación tienen el mismo valor para la propiedad **ConversationTopic** .</span><span class="sxs-lookup"><span data-stu-id="60897-119">Messages within the same conversation have the same value for the **ConversationTopic** property.</span></span> <span data-ttu-id="60897-120">El valor de esta propiedad suele ser la línea de asunto del mensaje inicial que generó la conversación.</span><span class="sxs-lookup"><span data-stu-id="60897-120">The value of this property is typically the Subject line from the initial message that spawned the conversation.</span></span>

<span data-ttu-id="60897-124">Durante el proceso de exportación de eDiscovery, se comparan estas tres propiedades para cada mensaje que coincida con los criterios de búsqueda.</span><span class="sxs-lookup"><span data-stu-id="60897-124">During the eDiscovery export process, these three properties are compared for every message that matches the search criteria.</span></span> <span data-ttu-id="60897-125">Si estas propiedades son idénticas para dos (o más) mensajes, esos mensajes se determinan como duplicados y el resultado es que solo se exportará una copia del mensaje si la desduplicación está habilitada.</span><span class="sxs-lookup"><span data-stu-id="60897-125">If these properties are identical for two (or more) messages, those messages are determined to be duplicates and the result is that only one copy of the message will be exported if de-duplication is enabled.</span></span> <span data-ttu-id="60897-126">El mensaje que se exporta se conoce como el "elemento de origen".</span><span class="sxs-lookup"><span data-stu-id="60897-126">The message that is exported is known as the "source item".</span></span> <span data-ttu-id="60897-127">La información sobre los mensajes duplicados se incluye en los informes **Results. csv** y **manifest. ## <a name="limitations-of-the-de-duplication-algorithm"></a><span data-ttu-id="60897-139">Limitaciones del algoritmo de desduplicación</span><span class="sxs-lookup"><span data-stu-id="60897-139">Limitations of the de-duplication algorithm</span></span>
