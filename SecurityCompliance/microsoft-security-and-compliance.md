# <a name="get-ready-for-the-new-microsoft-365-security-center-and-microsoft-365-compliance-center"></a>Prepararse para el nuevo centro de seguridad de Microsoft 365 y el centro de cumplimiento de normas de 365 de Microsoft

**Nos complace anunciar el [Centro de cumplimiento de normas de Microsoft 365](#microsoft-365-compliance-center), implantar iniciar al final de enero de 2019 y a través de marzo de 2019 de todas las novedades de [Centro de seguridad de Microsoft 365](#microsoft-365-security-center) y**. Lea este artículo para obtener información general sobre novedades, [qué esperar](#what-to-expect)y [licencias necesarias y los permisos necesarios](#required-licenses-and-permissions).

## <a name="microsoft-365-security-center"></a>Centro de seguridad de Microsoft 365

El nuevo centro de seguridad de Microsoft 365 incluye un panel para ayudarle a administrar y controlar la seguridad a través de identidades, datos, dispositivos, aplicaciones y la infraestructura. También tendrá un acceso sencillo a su puntuación todas las novedades de Microsoft seguro, nuevos informes de amenaza de dispositivo, nuevos informes de amenaza de identidad y los informes de seguridad de la aplicación en la nube. 

![Nuevo centro de seguridad de Microsoft 365](media/m365-security-center.png)

Al usar el centro de seguridad de Microsoft 365 por primera vez, verá información a través de la parte superior de la pantalla que le ayudarán a empezar a trabajar. También verá cómo navegar fácilmente a las características de seguridad que le interesa más exploración.

Una vez que el centro de seguridad de Microsoft 365 está habilitado para el inquilino, podrá tener acceso a él en [https://security.microsoft.com](https://security.microsoft.com). 

> [!NOTE]
> Debe asignar un rol de Azure Active Directory válido para acceder al centro de seguridad de Microsoft 365. Para obtener más información, vea la sección [requiere las licencias y permisos](#required-licenses-and-permissions) (en este artículo).

## <a name="microsoft-365-compliance-center"></a>Centro de cumplimiento de normas de 365 de Microsoft

El nuevo centro de cumplimiento de normas de Microsoft 365 proporciona visibilidad en Microsoft Administrador de cumplimiento, que refleja la situación de cumplimiento general y le proporciona acciones que le ayudarán a configurar las opciones para cumplir sus obligaciones complejo recomendadas. Tendrá un acceso sencillo a etiquetas de retención y de sensibilidad y las directivas, prevención de pérdida de datos (DLP), gobierno de datos, exhibición de documentos electrónicos, las solicitudes de datos asunto (valoración detallada), administración de casos y seguridad de la aplicación en la nube. Además, puede tener información modificable y aprovechar la automatización inteligente para reducir los riesgos de cumplimiento de normas y proteger su inmuebles digitales. 

![Centro de cumplimiento de normas de 365 de Microsoft](media/m365-compliance-center.png)

Al usar el centro de cumplimiento de normas de 365 de Microsoft por primera vez, verá información a través de la parte superior de la pantalla que le ayudarán a empezar a trabajar. Verá cómo navegar fácilmente a las características de cumplimiento que le interesa más exploración.

Una vez que el centro de cumplimiento de normas de Microsoft 365 está habilitado para el inquilino, va a tener acceso a él en [https://compliance.microsoft.com](https://compliance.microsoft.com).  

> [!NOTE]
> Debe asignar un rol de Azure Active Directory válido para acceder al centro de cumplimiento de normas de Microsoft 365. Para obtener más información, vea la sección [requiere las licencias y permisos](#required-licenses-and-permissions) (en este artículo).

## <a name="what-to-expect"></a>Qué esperar

### <a name="coming-soon"></a>Próximamente

El nuevo centro de seguridad de Microsoft 365 y el nuevo centro de cumplimiento de normas de Microsoft 365 están restaurando a partir de finales de enero y a través de marzo de 2019. Debe tener acceso al centro de cumplimiento de Microsoft 365 dentro de este período de tiempo y el nuevo centro de seguridad de Microsoft 365.

### <a name="easy-access"></a>¡Acceso sencillo!

Con navegación mejorada, soluciones integradas y una experiencia simplificada, podrá ver y tener acceso a la información que le interesa más y aprovechar las ventajas de la seguridad eficaces y soluciones de cumplimiento de normas en Microsoft 365.

### <a name="smooth-transition"></a>¡Transición gradual!

Puede esperar una transición sin problemas a los centros de nuevo. Después de este cambio se ha implantado totalmente, se planea retirar el anterior & de seguridad de Microsoft 365 centro de cumplimiento ([https://protection.microsoft.com](https://protection.microsoft.com)). Cambiará la experiencia del administrador, pero esto no afectar a la seguridad actual y las configuraciones de cumplimiento.

Después de esta actualización se implantado, si su organización tiene Microsoft 365 Enterprise E3 o E5, los administradores de seguridad y cumplimiento de normas pueden:

- Vaya directamente a [https://security.microsoft.com](https://security.microsoft.com) y [https://compliance.microsoft.com](https://compliance.microsoft.com); <br>o  
- Vaya al centro de administración de Microsoft 365 y, a continuación, navegue hasta el centro de cumplimiento de Microsoft 365 y nuevo centro de seguridad de Microsoft 365 (vínculos están en centros de administrador en el panel de navegación izquierdo).

> [!TIP]
> Si está utilizando el centro de cumplimiento de seguridad de Office 365 & ([https://protection.office.com](http://protection.office.com)), aún podrá configurar y administrar la configuración de Office 365 dentro de su & de seguridad de Office 365 existente centro de cumplimiento. Las configuraciones se mantendrá en el existente seguridad de Office 365 & centro de cumplimiento, así como en el nuevo centro de seguridad de Microsoft 365 y el centro de cumplimiento de normas de 365 de Microsoft.  

## <a name="required-licenses-and-permissions"></a>Permisos y licencias necesarias

Para obtener el nuevo centro de seguridad de Microsoft 365 y el centro de cumplimiento de normas de 365 de Microsoft, su organización debe tener una suscripción a Microsoft 365 E3 o E5 o una licencia por volumen equivalente (que consta de Office 365 Enterprise E3 o E5, movilidad de la empresa + seguridad E3 o E5 y Windows 10 Enterprise E3/E5).

Los usuarios deben tener asignados el rol de administrador Global, Administrador de cumplimiento de normas, Administrador de seguridad o seguridad lector en Azure Active Directory para tener acceso al nuevo centro de seguridad de Microsoft 365 o el centro de cumplimiento de normas de Microsoft 365.

- Pueden tener acceso a los administradores globales el centro de seguridad y el centro de cumplimiento

- Los administradores de cumplimiento pueden acceder al centro de cumplimiento

- Los administradores de seguridad o seguridad lectores pueden acceder al centro de seguridad

> [!NOTE]
> Funciones adicionales, incluidos los operadores de seguridad y Administrador de datos de cumplimiento, son muy pronto.
