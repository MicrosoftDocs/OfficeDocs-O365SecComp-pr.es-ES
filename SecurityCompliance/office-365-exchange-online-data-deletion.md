## <a name="continuous-replication"></a><span data-ttu-id="8d272-186">Replicación continua</span><span class="sxs-lookup"><span data-stu-id="8d272-186">Continuous Replication</span></span>

Replicación continua aprovecha la compatibilidad con la recuperación de Exchange Server base de datos intensivo para proporcionar tecnología que lleva a cabo la actualización asincrónica de uno o más copias de una base de datos de buzón de correo.
